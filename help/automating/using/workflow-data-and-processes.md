# Workflow data and processes{#workflow-data-and-processes}

通过Adobe Campaign，您可以导入、转换、过滤、管理和导出数据。可以执行这些操作来设计和自动化业务流程。

* 更新数据库：安装包或从文件上传数据。
* 流程自动化：通过工作流程，您可以设计自己的流程来转换数据、更新数据库、发送消息等。
* 导出数据：在单独的文件中导出包或数据集。
