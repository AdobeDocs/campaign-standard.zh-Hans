# 工作流数据和流程{#workflow-data-and-processes}

借助Adobe Campaign，您可以导入、转换、过滤、管理和导出数据。 这些操作可用于设计业务流程并实现业务流程自动化。

* 更新数据库：安装包或从文件上传数据。
* 流程自动化：通过工作流，您可以设计自己的流程来转换数据、更新数据库、发送消息等。
* 导出数据：将包或数据集导出到单独的文件中。
