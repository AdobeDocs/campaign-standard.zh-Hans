# 配置准则 {#about-configuration-guidelines}

## 兼容的浏览器{#compatible-browsers}

Adobe Campaign标准版兼容大多数计算机浏览器以及平板电脑的主要操作系统。

Adobe Campaign标准版仅支持平板电脑上的横向显示模式。

以下是与列表标准兼容的浏览器的Adobe Campaign:

**计算机：**

* Chrome（最新版本）
* Firefox（最新版本）
* Safari（最新版本）
* Microsoft Edge（最新版本）

**平板电脑：**

* iOS（版本7及更高版本）
* Android（4.4版和更高版本）
