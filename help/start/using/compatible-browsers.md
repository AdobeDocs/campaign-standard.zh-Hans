# Compatible browsers{#compatible-browsers}

Adobe Campaign Standard与大多数计算机的浏览器以及平板电脑的主要操作系统兼容。

Adobe Campaign Standard仅在平板电脑上支持横向显示模式。

以下是与Adobe Campaign Standard兼容的浏览器列表：

**计算机：**

* Chrome(最新版本)
* Firefox(最新版本)
* Safari(最新版本)
* Microsoft Edge(最新版本)

**平板电脑：**

* iOS(版本及更高版本)
* Android(版本4.4及更高版本)
