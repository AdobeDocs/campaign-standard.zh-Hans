# 关于扩展API{#about-extending-the-api}

此用例详细介绍了公共Profiles API中自定义资源配置文件的扩展字段。 例如，使用从计费系统生成的促销代码即时更新客户档案。

要发布API，您需要具有该角 **[!UICONTROL Datamodel]** 色。

* 扩展配置文件自定义资源
* 发布扩展
* 验证扩展字段是否在API中可用
