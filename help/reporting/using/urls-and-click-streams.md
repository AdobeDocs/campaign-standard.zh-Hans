# URL 和点击流{#urls-and-click-streams}

的 **URL和点击流** 显示在投放期间点击次数最多的URL，或者如果多个投放链接到营销策划或项目，则显示多个投放。

![](assets/delivery_reports_8.png)

每个表都由概要数字和图表表示。 您可以更改详细信息在各自的可视化设置中的显示方式。

的 **访问次数最多的链接** 表包含每个投放的收件人行为的可用数据，例如：

* **单击**:内容在投放中的点击次数。
* **独特点击**:点击投放内容的收件人数量。
