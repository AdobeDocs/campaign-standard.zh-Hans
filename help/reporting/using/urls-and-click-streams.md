# URL 和点击流{#urls-and-click-streams}

URL **和单击流** ，显示投放中点击次数最多的URL，或者如果投放链接到活动或项目，则显示多个URL。

![](assets/delivery_reports_8.png)

每个表由摘要编号和图表表示。 您可以更改详细信息在其各自的可视化设置中的显示方式。

“访 **问次数最多** ”链接表包含每个投放的收件人行为的可用数据，如：

* **单击**:在投放中单击内容的次数。
* **唯一点击**:单击收件人中内容的投放数。
