# 投诉{#complaints}

此 **[!UICONTROL Complaints]** 报告会将收到最多声明的交付内容识别为垃圾邮件。

![](assets/delivery_reports_complaints.png)

此 **失败** 此表按收件人域排序，显示已声明电子邮件或垃圾邮件的收件人数量。 该表的结果也以图表和概要数字显示。

此 **交付与阻止列表** 表列出了将电子邮件声明为垃圾邮件或垃圾邮件的收件人数量。 该表按投放排序。
