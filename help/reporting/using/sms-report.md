# 短信报告{#sms-report}

**SMS**&#x200B;报告提供有关SMS投放的详细信息，如提交率和跳出率。

![](assets/dynamic_report_sms.png)

**SMS摘要**&#x200B;表、图形和摘要编号包含已发送的SMS投放的可用数据。

* **已处理/已发送**:已发送的短信数。
* **已交付**:已发送的SMS数量。
* **跳出次数+错误**:无法传递的消息数。
