# 短信报告{#sms-report}

SMS报 **告提供SMS** 交付的详细信息，如交付率和弹出率。

![](assets/dynamic_report_sms.png)

SMS **摘要表** 、图形和摘要编号包含已发送的SMS发送的可用数据。

* **已处理／已发送**:发送的SMS数。
* **交付**:发送的SMS数量。
* **弹回次数+错误**:无法传递的消息数。
