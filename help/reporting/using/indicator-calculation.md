# 指标计算{#indicator-calculation}

>[!NOTE]
>
>为了更好地处理和管理大容量和实时分析，动态报告使用近似聚合来进行不同的计数估计。 近似聚合优惠有界的内存使用，并且通常比精确计算快。

下表根据投放类型为您提供了不同报表中使用的指示器的列表及其计算公式。
