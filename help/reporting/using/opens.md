# 打开次数{#opens}

此 **[!UICONTROL Opens]** 报告可识别收件人查看次数最多的交付内容。

![](assets/delivery_reports_opens.png)

* 浏览器
* 设备
* 平台
* 域

此 **[!UICONTROL TOP 5]** 表格和图形可显示消息数量最多的交付内容。
